O Tenda Form é ideal para qualquer tipo de consulta ao público. Nesta página vamos te orientar a montar uma estrutura igual a usada no museu itinerante Ciências sob Tendas. Não que este seja o único caso de uso, é apenas um ponto de partida.

Infraestrutura

No Ciências sob Tendas queria-mos oferecer ao usuário um brinde por responder a pesquisa, que no caso era uma "selfie" sobre uma moldura estilizada para o projeto. Essa funcionalidade está presente no código, porém, sua configuração será melhor detalhada em outro artigo. No entanto, para usar de tal recurso, contava-mos em nossa infraestrutura com uma impressora para imprimir as imagens.

A configuração da impressora é feita diretamente no computador servidor, não há configuração de impressora no Tenda Form. O servidor é o único dispositivo no qual o programa precisa estar instalado, pois é nele que irão se concentrar todos os serviços necessários para o funcionamento do Tenda Form.  Como servidor, usamos um netbook asus rodando sistema operacional Ubuntu 14.04. Os serviços instalados para o funcionamento correto do programa são:

  • MongoDB
  • Apache2
  • PHP5

O processo de instalação desses serviços serão melhor detalhados adiante.

Como várias pessoas poderiam ter interesse em responder a pesquisa, disponibilizamos dois tablets android como pontos de resposta. Um navcity e um galaxy tab 2. Nesse caso, como é mais de um ponto de resposta espalhado no ambiente, foi necessário o uso de um roteador wireless para criar uma rede local. Nessa rede, é importante que o computador servidor esteja com um IP fixo alocado. Pois dessa forma a configuração e manutenção dos pontos de resposta, neste caso são os tablets, fica muito mais fácil.

Instalação do Tenda Form

No ubuntu, o processo de instalação do Tenda Form é feito pela interface de comando de texto. Não é nada dificil, explicarei tudo direitinho, e será até divertido! Primeiramente abra o Unity Shell, clicando no botão ubuntu no canto superior esquerdo, ou com o "botão do windows" do seu teclado. No Unity Shell procure pelo programa "Terminal".

No terminal, instala os serviços necessários para o funcionamento do Tenda Form. Para agilizar, é o comando abaixo:

sudo apt-get install php5 apache2 mongodb libapache2-mod-php5 php5-mongo git

Após todos os serviços instalados, é hora de instalar o Composer. Basta usar o seguinte comando:

curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin -- --filename=composer

Nosso terceiro passo agora é baixar o código do Tenda Form. O comando para isso é:

git clone https://github.com/vsodre/tenda-form

Por fim, instalamos o programa:

composer install -d tenda-form/
